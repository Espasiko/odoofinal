# Plan de Mejora Manusodoo-Roto - 16/07/2025

## Análisis de Archivos y Recomendaciones

Tras un análisis exhaustivo de los archivos del proyecto, se han identificado áreas de mejora y optimización para el sistema de importación de Excel y procesamiento OCR de facturas.

### Archivos Críticos y Activos

- **`main.py`**: Punto de entrada principal de FastAPI con rutas para productos, proveedores, OCR y facturas.
- **`index.html`**: Página principal del frontend React con configuración de seguridad y conexiones permitidas.
- **`formatea_excel_nevir.py`**: Herramienta esencial para transformar archivos Excel de NEVIR al formato estándar.
- **`addons/barcode_scanning_sale_purchase`**: Módulo activo de Odoo para escaneo de códigos de barras en ventas y compras.
- **`check_fastapi.sh`**: Script útil para monitoreo y diagnóstico del contenedor FastAPI.
- **`listar_modulos_odoo.py`**: Script para verificar módulos instalados en Odoo.

### Archivos Parcialmente Útiles o de Referencia

- **`odooClient.ts`**: Cliente TypeScript para comunicación con Odoo, posiblemente reemplazado pero útil como referencia.
- **`odoo_suppliers.txt` y `db_suppliers.txt`**: Listas de proveedores con sus IDs, contienen información duplicada.

### Archivos Temporales o Inactivos

- **`project.db`**: Base de datos SQLite, posiblemente usada para almacenamiento temporal o caché.
- **`cookies.txt`**: Archivo temporal que almacena cookies de sesión para Odoo.

## Recomendaciones Técnicas

### 1. Consolidación y Limpieza de Archivos

- **Unificar archivos de proveedores**: Mantener solo uno de los archivos (`odoo_suppliers.txt` o `db_suppliers.txt`) y actualizarlo con información completa.
- **Revisar el cliente Odoo**: Determinar si `odooClient.ts` está en uso o si ha sido reemplazado por otro componente en el frontend.
- **Gestionar archivos temporales**: Añadir `cookies.txt` y posiblemente `project.db` al `.gitignore` si no son esenciales.
- **Mover scripts de utilidad**: Considerar la creación de un directorio `/utils` o `/scripts` para organizar los scripts de utilidad como `formatea_excel_nevir.py` y `listar_modulos_odoo.py`.

### 2. Mejoras en el Proceso de Importación

- **Generalizar scripts de formateo**: Extender `formatea_excel_nevir.py` para soportar más proveedores además de NEVIR.
- **Implementar validación de códigos de barras**: Utilizar el módulo `barcode_scanning_sale_purchase` para validar códigos de barras durante la importación.
- **Mejorar detección de duplicados**: Implementar verificación robusta de duplicados antes de crear nuevos productos.
- **Optimizar procesamiento por chunks**: Revisar y optimizar el mecanismo de rate limiting en la importación de Excel.

### 3. Mejoras en el Procesamiento OCR

- **Optimizar prompts por proveedor**: Ampliar la biblioteca de prompts específicos para mejorar la precisión OCR.
- **Implementar validación post-OCR**: Añadir validaciones específicas para NIF/CIF, códigos de producto y precios.
- **Mejorar la interfaz de corrección**: Facilitar la corrección manual de datos extraídos incorrectamente.
- **Ampliar sistema de caché OCR**: Optimizar el sistema de caché para reducir procesamiento redundante.

### 4. Integración Frontend-Backend

- **Estandarizar formato de respuestas API**: Asegurar que el backend devuelva `categ_id` como array `[id, nombre]` y no solo como ID numérico.
- **Garantizar inclusión de `seller_ids`**: Asegurar que todos los productos importados incluyan información completa de proveedores.
- **Implementar redirección post-importación**: Añadir navegación automática a la página de productos tras una importación exitosa.
- **Mejorar feedback visual**: Optimizar la visualización de progreso y resultados durante la importación.

### 5. Documentación y Mantenimiento

- **Actualizar README.md**: Mantener actualizada la documentación principal con instrucciones de instalación y uso.
- **Documentar scripts de utilidad**: Añadir más comentarios y documentación a los scripts para facilitar su uso y mantenimiento.
- **Crear diagramas de flujo**: Documentar los flujos de importación de Excel y procesamiento OCR con diagramas de secuencia.
- **Implementar tests automatizados**: Desarrollar pruebas para validar la integridad del sistema tras actualizaciones.
